# AI Sales Team Roadmap

## Phase 1, immediate acceleration

### 1. Sales Ops daily production loop
- Run Machining pipeline hygiene checks on schedule
- Produce prioritized manager and AE digests
- Keep writebacks recommend-only during trust-building

### 2. CNC throughput engine
- Prioritize leads with usable or recoverable websites
- Use search recovery when homepage data is weak
- Push ambiguous cases into review queue
- Promote clear disqualifications after the first reviewed tranche

### 3. Operational dashboard
- Show team health, queue health, findings pressure, owner pressure, and blockers
- Surface immediate actions rather than only raw counts
- Become the shared control room for Sales Ops and regional AI BDRs

### 4. AI BDR routing and enrichment
- Normalize inbound rows
- Route to the correct regional agent
- Assign AE owner from country/state
- Add dedupe and enrichment stages
- Emit a clean handoff note payload

### 5. Deployment and release discipline
- Keep code in this repo
- Add environment examples, deployment docs, and runbooks
- Ship in small, reviewable commits
